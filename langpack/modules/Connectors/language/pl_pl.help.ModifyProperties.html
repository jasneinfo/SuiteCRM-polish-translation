Konfiguracja właściwości łącznika


Użyj tej strony, aby skonfigurować adresy URL i klucze API dla łączników.

Na tej stronie są wyświetlone zakładki dla łączników, jakie dostarcza aplikacja.
Zakładki dla dowolnych innych łączników, które zainstalujesz, również wyświetlą się na tej stronie.

Możesz ustawić następujące parametry konfiguracji łączników:

  • Adres URL. Jeśli adres URL łącznika zmienił się - wprowadź nowy.

    The Endpoint URL field on the Hoover's tab displays the URL to the actual location to which soap calls are made.
  • Klucze API. Musisz posiadać klucze API, aby używać łączników do Jigsaw i Hoovers. Aplikacja Sugar dostarcza ships with wstępne klucze API dla obu łączników.
  • Maksymalna liczba na liście wyników. Aby zmienić liczbę wyników wyświetlanych na stronie, wprowadź pożądaną liczbę. Wartość domyślna to 20 rekordów na stronę.

Kliknij Zachowaj, aby zapisać zmiany ustawień. Kliknij Skasuj aby opuścić formularz bez zapisywania ustawien.
